# SAM — Japan Macro and Carry-Transmission Agent

SAM maintains a persistent view of Japan macro, sovereign rates, central-bank policy, currency dynamics, institutional flows, and the channels through which those forces can transmit into other markets.

SAM reasons in scenario-weighted distributions and records how predictions resolve. RED's challenge of one SAM thesis is the repository's principal governed-belief-revision case.

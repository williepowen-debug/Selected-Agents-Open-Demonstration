# RED Example State

> Demonstration only. The claim and sensors below are fictional.

## Challenge `DEMO-RED-01`

| Field | Example |
|---|---|
| Target claim | Sensor A and Sensor B independently confirm Mechanism M |
| Steelman | Both observations moved in the predicted direction before the outcome |
| Survival bar | The two observations must have distinct causal roots and sources |
| Counter-evidence | Both descend from Event Q and one original report |
| Strength | `STRONG` |
| Verdict | `BROKEN` as an independence claim; mechanism remains plausible |
| Concession required | Count A and B as one evidentiary leg |
| Confidence | `HIGH` → `MEDIUM` |
| Flip condition | A separately sourced observation tied to a different antecedent |

## Downstream effect

RED routes the revised evidence count to NEXUS and the target owner. PROME receives no action request because the preregistered two-leg requirement is no longer satisfied.
